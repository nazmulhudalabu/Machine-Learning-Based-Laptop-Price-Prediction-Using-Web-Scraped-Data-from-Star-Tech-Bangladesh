This code builds a complete machine learning pipeline to predict laptop prices by first scraping data from a Bangladeshi e-commerce site, extracting details like title, price, RAM, storage, and processor, then cleaning and preprocessing the data (fixing brand names, handling missing values, creating features like CPU family, and transforming price using log), followed by exploratory data analysis to understand distributions and relationships; after that, it selects important features, splits the dataset into training and testing sets, and trains two models—Linear Regression and Random Forest—with cross-validation, evaluates their performance using metrics like RMSE, R², and MAE, compares the results, and finally saves a cleaned dataset ready for analysis or submission
